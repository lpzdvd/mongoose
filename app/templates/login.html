Mongoose: un espacio para profesores y alumnos

Esta aplicación web forma parte de un Trabajo Fin de Grado orientado al diseño de bases de datos NoSQL que pretende ser un ejemplo sobre cómo este tipo de almacenes de datos pueden utilizarse como alternativa de éxito en el desarrollo de todo tipo de proyectos.

La base de datos

El componente indispensable de la aplicación es MongoDB, una base de datos NoSQL opensource apoyada por una extensa comunidad y que ofrece al desarrollador una excelente documentación en línea.

El lenguaje

Todo el desarrollo esta codificado en Python, un lenguaje de programación interpretado de propósito general cuya filosofía de diseño se orienta principalmente a la legibilidad del código.

El diseño

Intuitivo, flexible y potente, son las cualidades que han hecho de Bootstrap la herramienta ideal para el diseño de cualquier aplicación web independientemente de su tamaño o complejidad.

El servidor

Para desplegar la aplicación se ha optado por Flask, un microframework con licencia BSD inspirado en Werkzeug (una libreria de utilidades WSGI para Python) que destaca por ser tremendamente minimalista.

Presentación

La información mostrada al usuario se estructura con Jinja2, un lenguaje de plantillas HTML inspirado en Django que permite reutilizar partes del código y manipular dinámicamente el contenido de la página.

Una capa ODM

Dado el carácter documental de MongoDB, el uso de una capa ODM (Object-Document Mapper) como es el caso de Mongoengine facilita la interacción de los modelos como documentos de la base de datos.
